## Texin : Pin It. Recall It. 
A browser extension to quickly pin important texts and quickly scroll to them with ease.

## How to use :

1. Clone the repo in your folder
` git clone git@github.com:seika-afk/Texin.git`
2. Move zip file to another folder
3. Extract the zip file.
4. Go to Browser extension manager.
5. Turn on Developer mode
6. Press load unpacked Option on left corner.
7. Choose the folder , where you extracted the zip file.
8. Continue
9. Now the extension will be available.
